# my_projects

Guarda aqui los proyectos del Kanban como archivos `.json`.

Flujo recomendado:

1. Abre `index.html` con doble clic.
2. Usa **Cargar JSON** y selecciona uno o varios archivos de esta carpeta.
3. Edita proyectos y tareas desde la app.
4. Usa **Guardar proyecto** o **Guardar todos**.
5. Reemplaza en esta carpeta los JSON descargados por el navegador.

La app es HTML, CSS y JavaScript estatico. Por seguridad, los navegadores no permiten que una pagina
abierta por doble clic sobrescriba archivos locales automaticamente en todos los equipos.
